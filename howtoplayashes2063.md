This is an adapted version of the ["Original Docs"](https://www.gamersonlinux.com/forum/threads/ashes-gzdoom.5179/) for ArchLinux

- Install zmusic (ZMusic Only for ArchLinux: `git clone https://aur.archlinux.org/zmusic.git` -> `cd zmusic` -> `makepkg -si`)
- Intall GZDoom (ArchLinux: `git clone https://aur.archlinux.org/gzdoom.git` -> `cd gzdoom` -> `makepkg -si`)
- Run GZDoom one time to populate the necessary folders in your home
- GZDoom will error, close it
- Download only freedoom wad files [here](https://freedoom.github.io/download.html), OR Download All Files (Included the freedoom wad ones) [here](https://drive.google.com/drive/folders/1Tb8lpc2YcrCO8J90w5aB3Y3Evp0JETrD?usp=sharing)
- Extract and Paste All files in the following directory: /home/username/.config/gzdoom
- Give permissions to ashes2063 file with `chmod +x ashes2063` and Exec the ashes2063 file with: `./ashes2063` for playing Ashes 2063 on Linux with GZDoom :)
